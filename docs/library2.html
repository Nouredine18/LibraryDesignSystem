Typed.js Example

Using Typed.js

Typed.js is a library that types out text in an animated way.


This is how to use it: 


            <script src="https://cdn.jsdelivr.net/npm/typed.js@2.0.12"></script>
<script>
  var typed = new Typed('#typed-text', {
    strings: ["Here you can see some typed text.", "Typed.js makes it easy to add typing animations."],
    typeSpeed: 50
  });
</script>
