Lightbox

Je kan de Lightbox Library gebruiken door de CSS en JS files via een CDN te linken met je HTML file.

Deze files kun je hier vinden door op deze href link te klikken.
Eenmaal je erop hebt geklikt moet je naar "Download On Github" gaan.

Je download deze file, en sleept de `lightbox.css` en `lightbox-plus-jquery.js` files in jouw project. Eenmaal je dit hebt gedaan, link je de CSS en JS bestanden in je HTML file zoals hieronder weergegeven.

Bijvoorbeeld, de onderstaande code laat zien hoe je een afbeelding kan laten openen in een Lightbox wanneer erop wordt geklikt: 
      
        <a href="images/img_nature.jpg" data-lightbox="example-set" data-title="Nature and sunrise">
          <img src="images/img_nature.jpg" style="width:100%" class="hover-shadow cursor">
        </a>
Dit is hoe je gebruik maakt van deze library.
